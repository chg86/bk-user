# 蓝鲸用户管理 bk-user 前端

## 使用

``` bash
# 本地运行蓝鲸前端开发脚手架的文档工程
npm run doc

# 构建蓝鲸前端开发脚手架的文档工程
npm run build:doc

# 本地开发，创建 index-dev.html 修改文件内具体内容，指向自己的 SaaS 开发环境
cp index.html index-dev.html
npm run dev

# 创建 `Dll bundle`
npm run dll

# 打包构建
npm run build

# 打包构建分析
npm run build:analyzer
```
